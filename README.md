# Food Allergy Detection System

## Overview
The Food Allergy Detection System is a simple Java application that helps users identify common food allergens based on the food they consumed. It prompts the user to enter the food they ate and checks if it matches any of the common food allergens.

## Features
- Prompt user input: Asks the user to input the food they consumed.
- Allergen detection: Checks if the entered food matches any of the common food allergens.
- Warning message: Displays a warning message if the consumed food is a common allergen.
- User-friendly interface: Simple and easy-to-understand interface for users.

## How to Use
1. Clone this repository to your local machine.
2. Navigate to the directory where the code is located.
3. Compile the Java file using `javac FoodAllergyDetectionSystem.java`.
4. Run the compiled program using `java FoodAllergyDetectionSystem`.
5. Follow the on-screen instructions to enter the food you consumed.

## Requirements
- Java Development Kit (JDK) installed on your system.

## Example Usage

```

$ java FoodAllergyDetectionSystem
Enter the food you ate: Peanuts
Warning: The food you ate is a common allergen.

```
